# CP1_Leonardo-Csharp

## Descrição do Projeto
Este é um sistema de lanchonete virtual desenvolvido em C# como parte da atividade 3ESA do dia 20/03/2025. O programa permite:
- Cadastro de produtos
- Gerenciamento de pedidos (adicionar/remover itens)
- Cálculo de valores com descontos
- Visualização do pedido final

## Como Executar
1. Certifique-se de ter o .NET instalado (versão 8.0 ou superior)
2. Clone o repositório
3. Abra o terminal na pasta do projeto
4. Execute
